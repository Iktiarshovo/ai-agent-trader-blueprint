# VPS Setup Guide: AI-Agent Trader

This guide will show you step-by-step how to set up and run your AI-Agent Trader bot on a Linux VPS (Virtual Private Server).

**Prerequisites:**
- A VPS (Ubuntu 20.04 or 22.04 LTS is recommended)
- A Solana wallet (like Phantom or Solflare)
- All your API keys

---

### Step 1: Connect to Your VPS

Find your VPS IP address, username (usually `root`), and password from your VPS provider (like Aeza).

Open the Terminal (on macOS/Linux) or Command Prompt/PowerShell (on Windows) on your computer and run the following command:

```bash
ssh root@YOUR_VPS_IP_ADDRESS
